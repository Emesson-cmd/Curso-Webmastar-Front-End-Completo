Metatags para SEO

As metatags que ajudam na otimização dos motores de busca a encontrar seu site são:
  • description
  • keywords
  • author

Outros Atributos

  • title: quando você passa o mouse por cima aparece um texto explicativo.
  • alt: quando uma imagem não aparece, o "alt" será mostrado.
